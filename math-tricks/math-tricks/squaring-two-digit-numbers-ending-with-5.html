Trick to Square a Number Ending in 5

If a number ends in 5, follow this simple trick:

  • Let the number be N = 10x + 5, where x is the digit before 5.
  • Multiply x by x + 1.
  • Append 25 to the result.

Example:

25²: x = 2 → (2 × 3) = 6 → Answer: 625

35²: x = 3 → (3 × 4) = 12 → Answer: 1225
